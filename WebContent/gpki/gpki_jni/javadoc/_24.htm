■ 데이터 읽기

함수

int PKI_STORAGE_Read(int nType, char *pszPIN, unsigned char *pData,

                         int *pDataLen, char szErrorInfo[512]);

설명

저장매체로부터 데이터를 획득한다.

파라

메터

nType (input) : 읽어들일 데이터의 타입

pszPIN (input) : 스마트카드의 PIN 번호

pData (output) : 읽어들인 데이터의 포인터

pDataLen (output) : 읽어들인 데이터의 길이

szErrorInfo (output) : 에러발생 시 에러 정보

 

■ 데이터 저장

함수

int PKI_STORAGE_Write(int nType, char *pszPIN, unsigned char *pData,

                          int nDataLen, char szErrorInfo[512]);

설명

저장매체에 데이터를 저장한다.

파라

메터

nType (input) : 저장할 데이터의 타입

pszPIN (input) : 스마트카드의 PIN 번호

pData (input) : 저장할 데이터의 포인터

nDataLen (input) : 저장할 데이터의 길이

szErrorInfo (output) : 에러발생 시 에러 정보

메터
