에러코드

1000

ERR_ALREADY_INITIALIZED

설명

API가 이미 초기화 되어 있음

해결방법

1. 초기화 시, 환경변수(*ppCtx)의 값을 NULL로 설정하였는지를 확인한다.

2. 이미 초기화 함수를 호출하여 환경변수가 이미 초기화 되어있지 않은지 확인한다.

 

에러코드

1001

ERR_API_NOT_INITIALIZED

설명

API를 초기화해야 함

해결방법

GPKI_API_Init() 함수를 호출하여 API를 초기화 한 후, 사용하고자 하는 함수를 호출한다.

 

에러코드

1002

ERR_SET_WORK_DIR

설명

작업 디렉토리 경로 지정해야 함

해결방법

GPKI_API_Init() 함수 호출 시, 작업 디렉토리 경로를 지정한다.

 

에러코드

1003

ERR_MEM_ALLOC

설명

메모리 할당 실패

해결방법

해당 시스템에서 더 이상 메모리를 할당 할 수 없습니다. 물리적으로 메모리 사이즈를 늘려주거나, 사용하지 않는 메모리를 환원해 준다.

 

에러코드

1004

ERR_INSUFFICIENT_ALLOC_SIZE

설명

포인터에 할당한 메모리의 사이즈가 충분하지 않음

해결방법

포인터에 할당한 메모리 사이즈를 늘여준다.

1005

ERR_NO_ERR_MSG

설명

추가적인 에러 정보 없음

 

에러코드

1006

ERR_INVALID_INPUT

설명

올바르지 않은 인자 값

해결방법

함수의 구조대로 입력 값이 들어오지 않았을 때 발생한다. 에러 정보메시지에서 언급하는 입력 인자의 데이터를 확인한다.

 

에러코드

1007

ERR_NOT_SUPPORTED_FUNCTION

설명

해당 시스템에서 지원되지 않는 함수 임

해결방법

동적 라이브러리를 이용한 저장매체 지원은 Window 시스템에서만 가능하다.

 

에러코드

1008

ERR_TIME_OUT

설명

서버로 부터의 응답을 지정된 시간동안 기다렸으나 응답이 없음

해결방법

서버 담당 기관이나 표준보안API 배포기관에 문의한다.

기본적으로 응답을 기다리는 시간은 3분이며 이 시간이 짧거나 길 경우에는 환경파일 (gpkiapi.conf)의 “TIME_OUT” 값을 수정한다.
