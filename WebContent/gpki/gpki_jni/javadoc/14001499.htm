에러코드

1400

ERR_READ_CERT

설명

인증서를 저장매체에서 읽는데 실패

해결방법

1. 디스크에서 읽는 경우에는 해당 위치에 인증서 파일이 존재하는지, 해당 디렉토리의 읽기 권한이 있는지 확인한다.

2. 동적 저장매체를 이용하여 읽는 경우에는 저장매체가 해당 시스템과 연결되어있는지 저장매체의 PIN값을 올바르게 입력하였는지를 확인한다.

ERR_READ_PRIKEY

설명

개인키를 저장매체에서 읽는데 실패

해결방법

1. 디스크에서 읽는 경우에는 해당 위치에 개인키 파일이 존재하는지, 해당 디렉토리의 읽기 권한이 있는지 확인한다.

2. 동적 저장매체를 이용하여 읽는 경우에는 저장매체가 해당 시스템과 연결되어있는지 저장매체의 PIN값을 올바르게 입력하였는지를 확인한다.

1402

ERR_READ_FILE

설명

해당 경로의 파일을 읽는데 실패

해결방법

1. 해당 경로에 파일이 존재하는지 확인한다.

2. 해당 경로의 파일에 대한 읽기 권한이 있는지 확인한다.

 

에러코드

1403

ERR_SAVE_FILE

설명

해당 경로에 파일을 저장하는데 실패

해결방법

1. 해당 경로에 쓰기 권한이 있는지 확인한다.

2. 해당 경로에 동일한 이름의 디렉토리가 없는지를 확인한다.

설명

동적 라이브러리 위치를 지정한 곳에 동적 라이브러리가 존재하는지 확인한다.

 

에러코드

1409

ERR_FAIL_FREE_LIBRARY

설명

동적 라이브러리 해제 실패

해결방법

윈도우 용 함수 FreeLibrary()을 사용하는데 실패하였음. 에러 정보의 에러코드를 통하여 정확한 원인을 파악한다.

 

에러코드

1410

ERR_FAIL_LOAD_FUNCTION

설명

동적 라이브러리로부터 특정 함수 읽기 실패

해결방법

에러 메시지를 통하여 읽지 못한 함수 이름을 확인하고 로드한 동적 모듈에 해당 함수가 제공되는지를 확인하여 해당 함수가 없다면 표준보안API 배포기관을 통하여 해당 동적 모듈을 다시 제공 받는다.

 

에러코드

1411

ERR_DELETE_FILE

설명

매체에 저장되어있는 특정 파일을 삭제하는데 실패

해결방법

삭제하고자 한 파일이 디스크에 저장되어있는 파일이라면 해당 파일에 대한 접근 권한을 확인한다.

삭제하고자 한 파일이 스마트 카드에 저장되어있다면 에러 정보를 확인하여 정확한 에러 원인을 확인한다.

 

에러코드

설명

인증서를 저장하는데 실패

해결방법

저장하고자 한 위치가 스마트 카드라면 에러 메시지를 확인하여 정확한 원인을 학인한다.

ERR_FILE_CREATE

설명

파일을 생성하는데 실패

해결방법

에러코드

1420

ERR_FILE_SIZE_TOO_BIG

설명

파일의 크기가 처리할 수 있는 범위를 넘어섬

해결방법

대용량 파일 암호 처리를 위해서 지정한 원본 또는 암호 메시지의 크기가 너무 크다. 지정한 파일의 크기가 2G 이상인 경우에는 **_NoContent_File 로 끝나는 함수를 사용해야 한다.
