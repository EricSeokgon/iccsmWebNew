에러코드

1100

ERR_LOAD_LICENSE

설명

라이센스 파일 읽기 실패

해결방법

라이센스 파일 (gpkiapi.lic)이 API 초기화 시 지정한 작업 디렉토리에 있는지 확인한다.

 

에러코드

1101

ERR_WRONG_LICENSE

설명

잘못된 라이센스 파일

해결방법

라이센스 파일에 잘못된 부분이 있습니다. 라이센스 관리자에게 문의한다.

 

에러코드

1102

ERR_LICENSE_AUTHORITY

설명

라이센스 파일이 변조 됨

해결방법

라이센스 파일이 변조되었습니다. 라이센스 관리자에게 문의한다.

 

에러코드

1103

ERR_LICENSE_EXPIRED

설명

라이센스 파일이 만료 됨

해결방법

라이센스 파일이 만료되었습니다. 라이센스 관리자에게 문의하여 재 발급받는다.

 

에러코드

1104

ERR_SAVE_LICENSE_CACHE

설명

라이센스 캐시 파일 저장 실패

해결방법

에러메시지를 확인 하여, 캐쉬파일을 저장하려는 위치가 파일 쓰기가 가능한지 확인한다.

 

에러코드

1105

ERR_LICENSE_CHECK_FAILED

설명

라이센스 확인을 위해 시스템 정보를 확인하는데 실패함.

해결방법

에러 메시지를 확인하여 사용된 시스템 함수와 에러코드를 확인한다.

 

에러코드

1106

ERR_LICENSE_MODIFIED

설명

라이센스 파일이 수정 됨.

해결방법

사용하는 라이센스를 삭제하고 백업본을 사용한다.

 

에러코드

1107

ERR_LICENSE_NOT_FOR_GPKIAPI

설명

gpkiapi용 라이센스가 아님.

해결방법

API를 제공받은 곳을 통하여 gpkiapi용 라이센스를 발급받는다.

ERR_GEN_LICENSE_CACHE

설명

해결방법

시스템 오류일 수 있음. 재 시도 한다.
