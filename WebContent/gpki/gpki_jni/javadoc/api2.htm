기능

설명

인증서 정보 확인 모듈

X.509 인증서의 주요 필드 정보 확인

개인키 모듈

인증서 소유자의 중요한 정보인 개인키의 암호화/복호화

저장매체 모듈

인증서와 개인키를 저장매체로부터 읽어들이거나 저장, 삭제

유선용 전자 서명, 암호 메시지 생성/처리 모듈

유선 환경에서 보안 서비스를 제공하기 위해서 사용하는 보안 메시지 생성/처리

무선용 전자 서명, 암호 메시지 생성/처리 모듈

무선 환경에서 보안 서비스를 제공하기 위해서 사용하는 보안 메시지 생성/처리

시점확인 서비스 이용 모듈

시점확인 서버를 이용하여 특정 메시지에 대한 시점확인 토큰을 발급받음

인증서 내 정보를 이용한 본인 확인 모듈

인증서 내에 포함되어있는 정보를 이용하여 인증서 소유자에 대한 본인확인

보안 알고리즘 모듈

다양한 보안서비스 제공을 가능하게 하기 위한 보안 알고리즘

BASE 64 모듈

BASE64 인코딩 / 디코딩

디렉토리 접근 모듈

인증서와 인증서 폐지목록과 같은 디렉토리 서버에 게시되어있는 데이터 획득

객체 인증 모듈

상대방을 확인하기 위한 객체 인증 프로토콜
